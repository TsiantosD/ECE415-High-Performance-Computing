# High-Performance Computing (ECE415)

### Overview
- **CPU Optimization & OpenMP** (Labs 1 & 2):
Focuses on serial code optimizations (such as loop unrolling, loop fusion, and strength reduction) and multi-threading using OpenMP. Algorithms explored include Sobel edge detection and K-Means clustering.

- **GPU Programming with CUDA** (Labs 3, 4 & 5):
Covers CUDA C++ programming from basic kernel implementation to advanced GPU optimizations. Topics include shared memory utilization, asynchronous execution with streams, memory coalescing, thread coarsening, and multi-GPU execution. The algorithms accelerated on the GPU include 2D Image Convolution, Contrast Limited Adaptive Histogram Equalization (CLAHE), and N-Body astrophysics simulations.
